### Task 1: Clean up package.json dependencies

**Files:**
- Modify: `package.json`

**Step 1: Remove `sharp`, move the three experiment deps to devDependencies**

Open `package.json`. The `dependencies` block currently reads:

```json
"dependencies": {
  "@anthropic-ai/sdk": "^0.78.0",
  "sharp": "^0.34.5",
  "tree-sitter": "^0.21.1",
  "tree-sitter-typescript": "^0.23.2",
  "typescript": "^5.9.3"
},
"devDependencies": {
  "@types/node": "^25.3.2",
  "tsx": "^4.21.0"
}
```

Change it to:

```json
"dependencies": {
  "typescript": "^5.9.3"
},
"devDependencies": {
  "@anthropic-ai/sdk": "^0.78.0",
  "tree-sitter": "^0.21.1",
  "tree-sitter-typescript": "^0.23.2",
  "@types/node": "^25.3.2",
  "tsx": "^4.21.0"
}
```

`sharp` is deleted entirely — not moved. `typescript` stays in `dependencies` (runtime compiler).

**Step 2: Reinstall to update package-lock.json**

```bash

```

Expected: exits 0. `node_modules/sharp` should no longer exist.

**Step 3: Verify sharp is gone**

```bash

```

Expected: no output.

**Step 4: Verify the CLI still works**

```bash

```

Expected: help text prints normally (CLI has no import of sharp).

**Step 5: Commit**


### Task 2: Make bare `npx strand` run setup

**Files:**
- Modify: `src/cli/index.ts`

**Step 1: Change the no-command branch**

Find this block (lines 17–20):

```typescript
if (!command || command === "--help" || command === "-h") {
  printHelp();
  process.exit(0);
}
```

Replace with:

```typescript
if (command === "--help" || command === "-h") {
  printHelp();
  process.exit(0);
}

if (!command) {
  console.log(
    "No command given — running setup (generate + init) in current directory.",
  );
  console.log("Use 'strand --help' to see all commands.\n");
  await runSetup(undefined);
  process.exit(0);
}
```

**Step 2: Verify — no args runs setup**

```bash
npx tsx src/cli/index.ts
```

Expected output starts with:
```
No command given — running setup (generate + init) in current directory.
Use 'strand --help' to see all commands.

Setting up strand...
```

(It will proceed to scan cwd — that's fine, cwd is the strand repo itself. It will warn about no package.json — actually it will find one. It will generate a `.strand` file in `C:/dev/strand`. That's expected.)

**Step 3: Verify — `--help` still works**

```bash
npx tsx src/cli/index.ts --help
```

Expected: help text, no setup run.

**Step 4: Verify — unknown command still errors**

```bash

```

Expected: `Unknown command: badcmd` + help text, exit 1.

**Step 5: Rebuild**

```bash

```

Expected: exits 0.

**Step 6: Verify compiled version**

```bash

```

Expected: help text.

**Step 7: Commit**

